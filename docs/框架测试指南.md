# 框架测试指南

> 标准值源码：本地 `tests/helpers/system-core.mjs` → `SYSTEM_CORE_BASELINE`（**`tests/` 目录不入库**，clone 后需保留本地副本或从维护者处获取）。  
> 仅统计 `core/system-Core/`；其他 core 与全仓库加载日志**不计入**达标线。

**文档串联**：[README.md](../README.md) → [docs/README.md](README.md) → [system-core.md](system-core.md) | 发布前：[代码审查清单.md](代码审查清单.md) + [文档审查清单.md](文档审查清单.md)

## 命令

| 命令 | 达标 |
|------|------|
| `pnpm test:fast` | 59 pass，~1.3s |
| `pnpm test` | 64 pass |
| `pnpm test:e2e` | 3 pass |
| `pnpm test:all` | 含 e2e，以当次 CI 日志为准 |

环境：**Node ≥ 26**（`package.json` → `engines.node`）、`XRK_TEST=1`、`node --test --test-force-exit`；E2E 端口 `19000–19799`，就绪信号 `XRK_TEST_READY`。

测试引导（`tests/helpers/bootstrap.mjs`）：设置 `XRK_TEST=1`、stub `Bot`/`logger`，并 **import `src/bootstrap-globals.js`**（挂载 `plugin` / `segment`，否则 PluginsLoader 集成测会 `plugin is not defined`）。

## 标准值与实测（2026-06-14，Node ≥ 26）

| 类型 | 标准 | 断言 |
|------|------|------|
| HTTP | 12 | `module-inventory` + `loaders-integration`：`apis.has(key)`，key = `resolveCoreModuleKey`（如 `ai-workspace`，**无** `http/` 前缀） |
| stream | 7 | 同上 + `streams.has(name)` |
| plugin | 15 | `git ls-files core/system-Core/plugin/*.js`；vendor = **未 git 入库**的本地 `.js`（运行时仍会加载，不计入 baseline） |
| tasker | 4 | `module-inventory` |
| events | 4 | `module-inventory` |
| 快速单测（`pnpm test:fast`） | 59 | `package.json` → `test:fast` |
| 全量单测（`pnpm test`） | 64 | `package.json` → `test` |

**加载器（同次 `pnpm test` 日志，全仓库参考）**

| 指标 | 框架标准 | 日志参考 |
|------|----------|----------|
| ApiLoader | 12 key（system-Core，如 `ai-workspace`） | 全仓库模块数见启动日志（如 24） |
| StreamLoader | 7 流注册 | 全仓库流数见启动日志 |
| PluginsLoader | ≥15（入库） | 磁盘上可多于 baseline（含本地 vendor 插件） |

**E2E**

| 请求 | 状态 | 耗时 | 断言要点 |
|------|------|------|----------|
| `GET /api/plugins/summary` | 200 | 48 ms | `success`、`summary`、`plugins[]` |
| `GET /xrk/` | 200 | 12 ms | HTML |
| `GET /api/plugins/tasks` | 200 | 125 ms | `success`、`tasks[]` |

## 框架优越性（可复现）

| 维度 | 标准 + 实测 |
|------|-------------|
| 基准可审计 | system-Core 基准 + 快速单测 59 项；扩展 core 不稀释 **12/7/15/4/4** |
| CI 速度 | 全量单测 ~2.1 s（Node 26，本机 `pnpm test`） |
| 重依赖隔离 | `import` 后 `renderers.size === 0` |
| 测试离线 | `XRK_TEST=1` 跳过远程 MCP |
| 真启动 | E2E 子进程 `bot.run`，探活 12–125 ms |

复现：`pnpm test && pnpm test:e2e`。改标准值须同步 `system-core.mjs` 与本表。
