# 框架测试指南

> 标准值源码：`tests/helpers/system-core.mjs` → `SYSTEM_CORE_BASELINE`（**`tests/` 已入库**；CI 跑 `pnpm test:fast`）。  
> 仅统计 `core/system-Core/`；其他 core 与全仓库加载日志**不计入**达标线。

**文档串联**：[README.md](../README.md) → [docs/README.md](README.md) → [system-core.md](system-core.md) | 发布前：[代码审查清单.md](代码审查清单.md) + [文档审查清单.md](文档审查清单.md)

## 命令

入口：`tests/run.mjs`（`package.json` 各脚本只调此文件，勿再手写长文件列表）

| 命令 | 说明 |
|------|------|
| `pnpm test:fast` | 无 Bootstrap、无 Loader 集成（CI 快路径） |
| `pnpm test` | 除 e2e 外全部 `*.test.mjs` |
| `pnpm test:integration` | 仅 Loader 集成 |
| `pnpm test:e2e` | 真实启动 AgentRuntime |
| `pnpm test:all` | 含 e2e 的全部测 |

环境：**Node ≥ 26**、`XRK_TEST=1`；E2E 端口 `19000–19799`，就绪信号 `XRK_TEST_READY`。

测试引导（`tests/helpers/bootstrap.mjs`）：设置 `XRK_TEST=1`、stub `AgentRuntime`/`logger`，并 **import `src/bootstrap-globals.js`**（挂载 `PluginBase` / `msgSegment`，否则 PluginLoader 集成测会 `PluginBase is not defined`）。

## 标准值与实测（2026-06-14，Node ≥ 26）

| 类型 | 标准 | 断言 |
|------|------|------|
| HTTP | 12 | `module-inventory` + `loaders-integration`：`apis.has(key)`，key = `resolveCoreModuleKey`（如 `ai-workspace`，**无** `http/` 前缀） |
| stream | 7 | 同上 + `streams.has(name)` |
| plugin | 16 | `git ls-files core/system-Core/plugin/*.js`；vendor = **未 git 入库**的本地 `.js` |
| tasker | 4 | `module-inventory` |
| events | 4 | `module-inventory` |
| 快速单测（`pnpm test:fast`） | 见当次 `pnpm test:fast` 输出 |
| 全量单测（`pnpm test`） | 见当次 `pnpm test` 输出 |

**加载器（同次 `pnpm test` 日志，全仓库参考）**

| 指标 | 框架标准 | 日志参考 |
|------|----------|----------|
| HttpApiLoader | 12 key（system-Core，如 `ai-workspace`） | 全仓库模块数见启动日志（如 24） |
| AiStreamLoader | 7 流注册 | 全仓库流数见启动日志 |
| PluginLoader | ≥16（入库） | 磁盘上可多于 baseline（含本地 vendor 插件） |

**E2E**

| 请求 | 状态 | 耗时 | 断言要点 |
|------|------|------|----------|
| `GET /api/plugins/summary` | 200 | 48 ms | `success`、`summary`、`plugins[]` |
| `GET /xrk/` | 200 | 12 ms | HTML |
| `GET /api/plugins/tasks` | 200 | 125 ms | `success`、`tasks[]` |

## 框架优越性（可复现）

| 维度 | 标准 + 实测 |
|------|-------------|
| 基准可审计 | system-Core 基准 + 快速单测 59 项；扩展 core 不稀释 **12/7/15/4/4** |
| CI 速度 | 全量单测 ~2.1 s（Node 26，本机 `pnpm test`） |
| 重依赖隔离 | `import` 后 `renderers.size === 0` |
| 测试离线 | `XRK_TEST=1` 跳过远程 MCP |
| 真启动 | E2E 子进程 `bot.run`，探活 12–125 ms |

复现：`pnpm test && pnpm test:e2e`。改标准值须同步 `system-core.mjs` 与本表。
