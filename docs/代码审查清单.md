# 代码审查清单

自动化以 [框架测试指南.md](框架测试指南.md) 为准。

## 1. 自动化

```bash
pnpm test        # 59 pass，~3s
pnpm test:e2e    # 3 pass
```

## 2. 架构

| 检查项 | 期望 |
|--------|------|
| 业务在 `core/<name>/` | 不写 `src/` 业务 |
| HTTP / 插件 / 工作流 | 继承对应基类 |
| Tasker | 模块顶层不 `Bot.tasker.push` |
| 渲染器 | `import` 不启浏览器；截图前 `ensureLoaded()` |

## 3. 安全

| 检查项 | 期望 |
|--------|------|
| `/api/*` | 默认鉴权；`systemAuth: false` 才公开 |
| 文件路径 | `InputValidator` |

## 4. 配置三件套

`config/default_config/*.yaml` + commonconfig schema + 消费代码。

## 5. 启动冒烟

```bash
node app
```

打开 `/xrk`，确认截图/渲染可用。
