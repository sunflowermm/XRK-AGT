# MEMORY.md - 长期记忆（给助手的“可持续认知”）

仅在“主会话”中加载与更新：用于保存你提炼后的要点、决策与经验教训。

## 你应该写什么
- 值得长期保留的事实（例如：偏好、长期目标、重要约束）
- 重要决策与原因（为什么这样选、带来的收益/代价）
- 经验教训（以后遇到类似情况应该怎么做）

## 你不应该写什么
- 不要把临时讨论里的敏感细节直接塞进 MEMORY.md
- 不要把“未经证实的猜测”当成结论保存

## 更新规则（写清楚可验证内容）
- 当你得到新结论：补充“事实 + 来源/依据 + 对后续的影响”
- 当某条旧信息失效：删除或注明“已过期/已调整”

## 输出格式建议
- 用短条目：`- 结论：... / 依据：... / 影响：...`
